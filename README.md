# Decision Support System – Music Streaming Analytics

## Overview
This project focuses on the design and development of a Decision Support System (DSS) for a music streaming platform.

The objective is to transform raw data into structured insights to support business decision-making through data warehousing, OLAP analysis and dashboards.

---

## Data Processing

### Data Understanding & Cleaning
- Analysis of raw data
- Data cleaning and transformation
- Data enrichment

### Data Warehouse Design
- Dimensional modeling (fact and dimensions)
- Star/Snowflake schema design

### ETL Process
- Data extraction, transformation and loading using SQL Server Integration Services (SSIS)
- Integration of multiple data sources

---

## Analytical Layer

### OLAP Analysis
- Cube creation using SQL Server Analysis Services (SSAS)
- Aggregations, slicing and drill-down
- MDX queries for advanced analysis

### Dashboarding
- Creation of dashboards for business insights
- Visualization of key metrics and trends using Power BI

---

## Key Insights
- Analysis of music streaming patterns
- User behavior and content popularity
- Support for strategic business decisions

---

## Technologies
- SQL
- SQL Server Integration Services (SSIS)
- SQL Server Analysis Services (SSAS)
- Data Warehouse
- OLAP
